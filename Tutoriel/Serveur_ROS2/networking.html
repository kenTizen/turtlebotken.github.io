Configuration du Robot

Ces instructions permettent de configurer le PC utilisateur et le robot pour une communication de base.

La connexion au routeur n'est nécessaire que pour la première mise en service, Assurez-vous que votre Robot n'est connecté a aucun Routeur et qu'il s'agit bien de sa première mise en service

Le Robot

La première étape de la configuration du TurtleBot 4 lite consiste à l'allumer et à le connecter au réseau Wi-Fi du routeur que vous disposez.

Allumer le Robot

Placez le TurtleBot 4 lite sur sa station de recharge. Le voyant blanc du Robot clignote pendant quelques secondes et il s'allume (voyant blanc persisant hors charges). Laissez au robot le temps de démarrer et noté que lorsqu'il a fini démarrer il émet un son.

Connexion au routeur (première mise en service)

Au premier démarrage, le Raspberry Pi entrera en mode point d'accès (AP), ce qui vous permettra de vous connecter à lui par Wi-Fi. Sur votre PC, connectez-vous au réseau Wi-Fi Turtlebot4. Le mot de passe est également Turtlebot4.

Note

Le réseau TurtleBot 4 AP est un réseau 5GHz. Votre ordinateur doit prendre en charge le Wi-Fi 5GHz pour se connecter au réseau.

SSH dans la Rasberry Pi

Une fois connecté, vous pouvez vous connecter en SSH au Raspberry Pi pour configurer son Wi-Fi. Ouvrez un terminal sur votre PC et appelez :

ssh ubuntu@10.42.0.1

Connectez-vous en utilisant le mot de passe turtlebot4.

Connecté la Rasberry Pi à votre routeur

Une fois connecté, configurez le Raspberry Pi pour qu'il se connecte à votre réseau Wi-Fi en suivant la vidéo ci-dessous.

Astuce

Connectez le Raspberry Pi à un réseau Wi-Fi 5GHz pour des performances optimales.

    Dans votre session SSH, lancez TurtleBot 4 setup tool:

    turtlebot4-setup

    Ceci lancera l'outil de configuration de TurtleBot 4. Naviguez jusqu'au menu " Wi-Fi Setup " et configurez votre connexion. Lorsque vous avez terminé, enregistrez et appliquez les paramètres.

    Note

    Réglez votre mode Wi-Fi sur "Client" lorsque vous vous connectez à un réseau existant.

    Trouver la nouvelle Rasberry Pi

    Une fois les paramètres Wi-Fi appliqués, le Raspberry Pi redémarre et se connecte à votre réseau. Le DHCP du routeur lui attribue une nouvelle adresse IP.

    Vous pouvez verifier la nouvelle adresse IP de la Rasberry Pi via l'Interface du routeur

    Note

    Sur l'interface du routeur, pour 1 Robot on a deux adresses IP, celle du Robot commençant souvent par iRobot, et celle de la Rasberry Pi qui est toujours Ubuntu

    Il est indispensable d'être connecté au même routeur que le Robot afin de pouvoir lancez de nouvelles session SSH apres la mise en service (Mettez votre Pc sur la bande 5GHz du routeur)

    Accès au serveur web Create® 3

    Une fois que vous avez obtenu l'adresse IP du Raspberry Pi, vous pouvez accéder au serveur web Create® 3

    Ouvrez un navigateur web et naviguez vers l'IP de votre Raspberry Pi avec le port 8080.

    Vous serez accueilli par la page d'accueil du serveur web Create® 3.

    Serveur web iRobot® Create® 3 - Connexion

    La page de connexion du serveur web Create® 3 permet de changer le nom du robot et de le connecter à un réseau Wi-Fi 2,4 GHz.

    Souvenez-vous que le turtlebot4-lite a deux cartes wifi, celle du Robot et celle de la Rasberry Pi

    Le haut de la page de connexion indique l'adresse IPv4 actuelle du robot.

    La case "Mettre à jour les noms du robot" permet à l'utilisateur de modifier le nom d'hôte du robot et son nom Bluetooth (qui peut être utilisé pour le trouver dans iRobot Coding).

    La case "Se connecter à un réseau Wi-Fi 2,4 GHz" permet à l'utilisateur de se connecter à n'importe quel réseau que le robot peut percevoir. En cliquant sur la case à droite de "Tapez le nom de votre réseau Wi-Fi", vous obtiendrez une liste déroulante de tous les réseaux disponibles.

    Connetez le serveur Create® 3 à la bande 2.4GHz de votre routeur

    Mise à jour du TurtleBot 4 lite

    Il est recommandé de mettre à jour Create® 3 et le Raspberry Pi lors de leur première utilisation afin de bénéficier des dernières corrections de bugs et améliorations.

    Create® 3

    Vérifierz les versions du logiciel Create® 3 pour voir si une version plus récente du micrologiciel est disponible. Vous pouvez vérifier la version du firmware de votre Create® 3 en visitant le serveur web.

    Si un nouveau micrologiciel est disponible, téléchargez-le, puis accédez au serveur web. Allez dans l'onglet Mise à jour, téléchargez le micrologiciel, puis mettez à jour votre robot.

    Paquets Raspberry Pi

    SSH dans le Raspberry Pi, puis mettez à jour tous les paquets en appelant :

    sudo apt update && sudo apt upgrade
