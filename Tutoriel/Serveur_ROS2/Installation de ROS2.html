Installation de Ros 2

Les instructions suivantes vont permettre à l'utilisateur d'installer Ros 2

Assurez-vous que vous êtes sur votre machine virtuelle et ouvrez un Terminal

Définir une locale

Assurez-vous que vous avez une locale qui supporte UTF-8. Si vous êtes dans un environnement minimal (comme un conteneur docker), la locale peut être quelque chose de minimal comme POSIX. Nous testons avec les paramètres suivants. Cependant, cela devrait fonctionner si vous utilisez une locale différente supportant l'UTF-8. 


                            locale  # check for UTF-8

     sudo apt update && sudo apt install locales
     sudo locale-gen en_US en_US.UTF-8
     sudo update-locale LC_ALL=en_US.UTF-8 LANG=en_US.UTF-8export LANG=en_US.UTF-8
                                    
    locale  # verify settings

Configuration des sources

Vous devrez ajouter le dépôt ROS 2 apt à votre système. Assurez-vous d'abord que le dépôt Ubuntu Universe est activé.


     sudo apt install software-properties-common
     sudo add-apt-repository universe

Ajoutez maintenant la clé GPG de ROS 2 avec apt.


     sudo apt update && sudo apt install curl -y
     sudo curl -sSL https://raw.githubusercontent.com/ros/rosdistro/master/ros.key -o /usr/share/keyrings/ros-archive-keyring.gpg

Ajoutez ensuite le dépôt à votre liste de sources.

echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/ros-archive-keyring.gpg] http://packages.ros.org/ros2/ubuntu $(. /etc/os-release && echo $UBUNTU_CODENAME) main" | sudo tee /etc/apt/sources.list.d/ros2.list > /dev/null

Installer le paquet Ros2

Mettez à jour vos caches de dépôts apt après avoir configuré les dépôts.


    sudo apt update

Les paquets ROS 2 sont construits sur des systèmes Ubuntu fréquemment mis à jour. Il est toujours recommandé de s'assurer que votre système est à jour avant d'installer de nouveaux paquets.


    sudo apt upgrade

Installation sur le bureau (recommandée) : ROS, RViz, démos, tutoriels


    sudo apt install -y ros-humble-desktop

Installation de ROS-Base (Bare Bones) : Bibliothèques de communication, paquets de messages, outils de ligne de commande. Pas d'outils GUI.


    sudo apt install -y ros-humble-ros-base

Outils de développement : Compilateurs et autres outils permettant de créer des paquets ROS


    sudo apt install ros-dev-tools

Ajoutez ensuite les messages Create® 3 :


    sudo apt install -y ros-humble-irobot-create-msgs

Installez les paquets suivants


    sudo apt install -y build-essential python3-colcon-common-extensions python3-rosdep ros-humble-rmw-cyclonedds-cpp

Configuration de l'environnement

Recherche du script d'installation

Configurez votre environnement en créant le fichier suivant.


    # Replace ".bash" with your shell if you're not using bash
    # Possible values are: setup.bash, setup.sh, setup.zsh
    source /opt/ros/humble/setup.bash

À ce stade, nous allons définir notre RMW (middleware ROS 2) par défaut. Le RMW que vous définissez ici doit correspondre au RMW de votre robot, qui peut être trouvé sur la page de configuration de l'application. saisissez la ligne suivante:


    echo "export RMW_IMPLEMENTATION=rmw_cyclonedds_cpp" >> ~/.bashrc
